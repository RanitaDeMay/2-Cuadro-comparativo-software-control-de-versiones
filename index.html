Conclusión

Me parece que siempre es importante conocer alternativas de los diferentes software o servicios que vamos a utilizar. Siempre debemos conocer alguna opción antes de decidir usar algo o comprarlo. En este caso se compararon tres diferentes sistemas de control de versiones: GIT, Mercurial SCM y Fossil SCM. Las tres las pude encontrar gracias a la página que nos proporcionó el docente. Si bien no se abordaron a fondo las ventajas o desventajas que tiene cada una sobre otra, sí nos sirvió para tener otra opción antes cualquier situación que se presente. Yo creo que el docente nos asignó esta actividad para ampliar nuestro conociemiento sobre estos sistemas que muy posiblemente nos tocará usar en el ámbito laboral.
